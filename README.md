# BlazorShop
Simple shop application built with Blazor WebAssembly

[![Facebook](https://img.shields.io/badge/facebook-%231877F2.svg?style=for-the-badge&logo=Facebook&logoColor=white)](https://www.facebook.com) [![GitHub](https://img.shields.io/badge/github-%23121011.svg?style=for-the-badge&logo=github&logoColor=white)](https://github.com)

## :eyes: Try it out

Make sure you have the [.NET Core 3.1 SDK](https://dotnet.microsoft.com/download) installed on your machine. Then do:

```
git clone https://github.com/devguy201-9/E-Vegetable-shop.git
dotnet run --project BlazorShop.Web/Server/BlazorShop.Web.Server.csproj
```
